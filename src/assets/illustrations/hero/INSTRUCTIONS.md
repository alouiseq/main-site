# Hero Illustration Instructions

## Where to Get the Illustration

Visit **unDraw** (https://undraw.co/) and search for "developer" or "programming" themed illustrations.

## Recommended Illustrations:
- "Programming" - https://undraw.co/illustrations
- "Developer activity"
- "Code typing"
- "Working"

## How to Customize:

1. Click on any illustration you like
2. Click the color picker in the top right
3. Set the primary color to: **#00D4FF** (arc-reactor-blue)
4. Download as SVG
5. Save the file as `hero-illustration.svg` in this directory
6. The illustration will automatically be used in the HeroSection component

## Alternative:

If you prefer a different style, you can also use **Humaaans** (https://www.humaaans.com/) to create a custom character illustration.

## File to Replace:

Once downloaded, place your illustration here:
`src/assets/illustrations/hero/hero-illustration.svg`
